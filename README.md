# Marketing Portfolio

## Project Info

Alisa Chen's marketing portfolio built with Vite, React, TypeScript, shadcn/ui, and Tailwind CSS.

## Development

Install dependencies and start the dev server:

Follow these steps:

```sh
npm i
npm run dev
```

Build for production:

```sh
npm run build
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
